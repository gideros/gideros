Sprite:getSize

Available since version: Gideros 2016.12.1

Description:

Returns the width and height of the sprite, in pixels.

Syntax:

(number), (number) = Sprite:getSize()

Return value:

  • number: Width of the sprite.
  • number: Height of the sprite.
